# Code de Conduite

## Notre Engagement
Nous nous engageons à faire de la participation à notre projet une expérience exempte de harcèlement pour tout le monde.

## Nos Standards
* Utiliser un langage accueillant et inclusif.
* Être respectueux des points de vue divergents.
* Accepter les critiques constructives.

## Signalement
Les comportements inacceptables peuvent être signalés au mainteneur du projet.
